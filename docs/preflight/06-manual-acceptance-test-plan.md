> Setup baseline for all tests: Windows 11 PC, `npm install`, set `VERIDIAN_WATCH_DIR` to a real git repo, `TELEMETRY_POLL_MS=30000`, then `npm run dev`. Open http://localhost:3000.

1. On the Dashboard, confirm the initial HUD is **empty/unknown**, NOT "MIRA-VPN / auth.service.ts". (Expected after F-006 fix; currently shows fake seed.)
